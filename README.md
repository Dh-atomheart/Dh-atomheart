## 👋 Hi, I'm DorianYu

I'm an **AI Application Developer** focused on building practical, user-facing software around **LLMs, local-first workflows, desktop applications, and full-stack tools**.

I care about the full path from idea to product: turning LLM capabilities into real workflows, building clean interfaces, connecting AI models with local runtimes, databases, files, and cloud services, and shipping tools that solve concrete problems instead of staying as demos.
